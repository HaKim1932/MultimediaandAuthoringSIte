Animation

Rotoscoping

Frame-by-Frame Adobe Animate Tracing

Rotoscoping is a classical animation technique where artists trace over live-action footage frame by frame to produce fluid, realistic motion. In this project, I applied rotoscoping to a selected video clip, carefully tracing each frame to create a stylized animated version that preserves the natural fluidity of movement.

3D Rendering

3D Rendering — Solar System

Blender Texturing Lighting Animation

3D rendering is the process of converting three-dimensional digital models into two-dimensional images using simulated light, materials, and camera perspectives. This project builds the entire solar system from scratch — the Sun plus all eight planets, each with NASA-sourced textures and orbital animation.

How It Was Made

The Creative Process

01
🎯
Research & Planning
Each project began with deep research — studying the history of rotoscoping from its origins, and understanding the physics of planetary orbits before touching any software.
02
🌐
3D Modeling & Scene Building
Each planet was modeled as a UV sphere, UV-unwrapped to accept NASA texture maps. Orbital paths were set using curve constraints, and the Sun was given an emission shader.
03
🎨
Rotoscoping Frame by Frame
Importing video footage as reference, then meticulously drawing over each frame using a graphics tablet. Key poses identified first, then in-betweens filled in for smooth motion.
04
💡
Lighting & Texturing
A point light at the Sun’s center simulates solar radiation. Each planet surface uses diffuse, specular, and bump maps to achieve photorealistic depth and texture detail.
05
🎥
Rendering & Export
Final renders produced in Blender’s Cycles engine at HD resolution with GPU acceleration. Frames compiled into video sequences using the built-in video editor.
06
💻
Web Portfolio
Presented through this interactive portfolio site, built from scratch — canvas animations, celestial toggle, star-field, and responsive design. No frameworks, all vanilla.
